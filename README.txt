영어("en") 한국어("ko") 텍스트 인식 지원

요구사항
1. Python 3.8 이상
2. 프로젝트 디렉토리로 이동하여 필요한 패키지를 설치: pip install -r requirements.txt


실행 방법
Flask 서버 실행 명령어 : python app.py

API 사용
- 엔드포인트: `/ocr`
- HTTP 메서드: POST
- 요청: `image` 키로 이미지를 전송
- 지원되는 파일 형식: PNG, JPG, JPEG
- 파일 크기는 10MB를 초과할 수 없음


주의사항
- 개발 환경에서 실행시 `debug=True`
- 운영 환경에서 실행시 `debug=False`

문제 해결:
- "이미지 파일이 제공되지 않았습니다." 오류: 요청에 `image` 파일이 포함되어 있는지 확인
- "파일 크기가 10MB를 초과합니다." 오류: 파일 크기를 확인하고, 10MB 이하의 파일을 업로드
- "허용되지 않는 파일 형식입니다." 오류: PNG, JPG, 또는 JPEG 파일만 지원
- "이미지 처리 시간이 30초를 초과했습니다." 오류: 복잡한 이미지나 서버의 성능 문제
